Connecteur Adobe-AEM-Brightcove : Installation

Cette rubrique traite de l'installation du connecteur Brightcove pour Adobe Experience Manager (AEM).

Introduction

Le connecteur Adobe-AEM-BrightCove vous permet de gérer les vidéos et les lecteurs Brightcove Video Cloud dans AEM et d'intégrer facilement des vidéos dans des pages AEM.

Notez que ce guide utilise AEM 6.3. Le connecteur fonctionne également avec AEM 6.2, 6.4 et 6.5 et les étapes sont similaires.

Mise en route

Si vous n'avez pas encore d' instance de développement AEM, vous pouvez facilement en créer une à l'aide du fichier .jar de démarrage rapide d'AEM :

java  -Xmx1400M -jar AEM_6.3_Quickstart.jar

Si vous utilisez ce démarrage rapide, votre instance AEM sera hébergée à l'adresse suivante : http://localhost:4502 par défaut. Les sections suivantes prendront cet emplacement par défaut. Par conséquent, si vous utilisez un emplacement ou un port différent, vous devrez ajuster les URL affichées dans les étapes, ou utiliser le menu de navigation AEM à la place.

Installation du connecteur

Les étapes suivantes vous permettront d'installer le connecteur à l'aide de l'interface classique.

  • Allez sur https://github.com/brightcove/Adobe-AEM-Brightcove-Connector/releases/ et téléchargez le connecteur de votre version d'AEM (le fichier brightcove-all... zip).
  • Démarrez votre instance de développement d'AEM.
  • Après vous être connecté, accédez au Gestionnaire de packages CRX (http://localhost:4502/crx/packmgr/index.jsp)
  • Cliquez sur le bouton Télécharger :
  • Dans la boîte de dialogue Charger, accédez au fichier zip du connecteur et cliquez sur OK pour le télécharger.
  • Lorsque le package apparaît dans la liste, cliquez sur Installer pour l'installer :
  • Accédez à la page des lots de console (http://localhost:4502/system/console/bundles) pour vérifier que le package est actif :

Lorsque vous avez terminé ces étapes, le connecteur est installé avec succès et vous devez passer à la configuration.

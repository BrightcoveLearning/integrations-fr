Prise en charge de plusieurs langues

L'extension fournit la prise en charge de plusieurs langues d'interface utilisateur pour Business Manager Interface. Remarque : Cette version de l'extension prend en charge l'anglais et le français pour la langue de l'interface utilisateur.

Ajouter une vidéo

Voici les étapes à suivre pour ajouter des vidéos.

  1. Accédez à Outils marchands -> Brightcove -> Ajouter une vidéo
  2. Sélectionnez Langue dans le menu (les paramètres régionaux par défaut seront sélectionnés pour le site) :
  3. Cliquez sur l'option de sélection en regard du champ Vidéo.
  4. Dans la boîte de dialogue qui s'affiche, sélectionnez le fichier vidéo :
  5. Cliquez sur l'option Sélectionner en regard du champ Miniature, et une boîte de dialogue contextuelle similaire apparaîtra dans laquelle vous pouvez sélectionner le fichier miniature.
  6. Cliquez sur l'option Sélectionner en regard du champ Poster, et une boîte de dialogue contextuelle similaire apparaîtra dans laquelle vous pouvez sélectionner le fichier d'affiche.
  7. Soumettre le formulaire pour amorcer le processus d'ingestion :
  8. Valider le message de réussite sur le formulaire après la soumission :
  9. Vous pouvez vérifier que dans la vidéo a été ingérée correctement en vous connectant à Brightcove Studio et en accédant au module Media :
  10. Vous pouvez également cliquer sur la vidéo pour révéler les détails afin de vérifier le code régional :

Affecter des vidéos - Catégories

Nous allons d'abord examiner les étapes pour attribuer des vidéos aux catégories.

Affecter des vidéos à des catégories

  1. Accédez à Outils marchands -> Brightcove -> Attribuer une vidéo (catégories).
  2. Les vidéos sont affectées aux paramètres régionaux.
    • Les données relatives aux paramètres régionaux peuvent être affichées en modifiant les paramètres régionaux à partir de la liste déroulante :

      Notez que les seules locales actuellement prises en charge sont l' anglais (États-Unis) et le français (France)

    • Les vignettes vidéo peuvent être visualisées en plaçant le curseur sur Video ID :
  3. Cliquez sur l'option Attribuer une vidéo pour affecter une ou plusieurs vidéos.
  4. Les vidéos déjà affectées apparaîtront comme sélectionnées sur la grille d'attribution de la vidéo :
  5. L'emplacement attribué ne peut pas être utilisé pour les autres vidéos pour les mêmes paramètres régionaux :
  6. Les joueurs et les emplacements peuvent être modifiés en utilisant les déroulants correspondants disponibles dans chaque tuile vidéo.
  7. Sélectionnez une autre vidéo avec la fente différente.
  8. Cliquez sur le bouton Enregistrer la vidéo pour attribuer les vidéos sélectionnées :
  9. Vérifiez la vidéo assignée et l'emplacement :
  10. Vérifiez les données de l'objet catégorie.
  11. Accédez à Outils marchands -> Catalogues.
  12. Sélectionnez le catalogue.
  13. Sélectionnez la catégorie/sous-catégorie.
  14. Sélectionnez Modifier.
  15. Sélectionnez Attributs de catégorie.
  16. Faites défiler la page jusqu'à la section Brightcove :
  17. Pour affecter des vidéos à des sous-catégories, cliquez sur la catégorie :
  18. La liste des sous-catégories apparaîtra :
  19. Suivez les étapes 3 à 16 pour affecter des vidéos à des sous-catégories.

Intégrer la vidéo

Les étapes vidéo d'intégration vous permettent d'obtenir du code pour rendre des vidéos sur n'importe quelle page ISML.

  1. Accédez à Outils marchands -> Brightcove -> Incorporer la vidéo.
  2. Sélectionnez la vidéo et le lecteur souhaités, puis cliquez sur Obtenir le code:
  3. Une boîte de dialogue contextuelle s'affiche avec le code. Sélectionnez le bloc de code et il sera copié dans le presse-papiers :
  4. Collez le code dans la ressource de contenu, l'emplacement de contenu ou dans n'importe quel modèle ISML pour rendre la vidéo.

Chargement en bloc

Le chargement en masse permet d'ingérer des vidéos en quantité groupée en téléchargeant des fichiers .csv. La tâche traite le ou les fichiers .csv et ingère les vidéos dans le compte Brightcove. Un exemple de fichier BulkUpload est joint dans le fichier zip sous méta répertoire.

Notez que le videofile, videoposter et videothumbnail doit être disponible sur des chemins accessibles au public.
  1. Accédez à Outils marchands -> Brightcove -> Chargement en bloc.
  2. Sélectionnez le fichier .csv.
  3. Cliquez sur Soumettre :
  4. Accédez à Administration -> Tâches.
  5. Sélectionnez Travail de chargement en bloc et cliquez sur Exécuter.
  6. Vérifiez le statut de fin de tâche.
  7. Vérifiez l'objet vidéo dans le compte Brightcove.
  8. Le dossier traité par Transfert groupé le travail sera déplacé vers BulkUpload_processed dossier sous Temp annuaire.

Catégorie

  1. Accédez à Outils marchands -> Brightcove -> Affectation en bloc.
  2. Sélectionnez Catégorie.
  3. Sélectionnez le fichier .csv.
  4. Cliquez sur Soumettre:
  5. Le fichier sera téléchargé dans le Temp répertoire sous le bulkAssignCatégorie dossier.
  6. Accédez à Administration -> Tâches.
  7. Sélectionnez le travail d' affectation en bloc et cliquez sur Exécuter.
  8. Vérifiez le statut de fin de tâche.
  9. Vérifiez la ou les vidéos assignées.
  10. Le fichier traité par le travail Bulk-Assigner sera déplacé vers le dossier BulkAssignCategory_Processed sous le répertoire Temp.

Produit

  1. Accédez à Outils marchands -> Brightcove -> Affectation en bloc.
  2. Sélectionnez Produit.
  3. Sélectionnez le fichier .csv.
  4. Cliquez sur Soumettre:
  5. Le fichier sera téléchargé dans le Temp répertoire sous le bulkAssignProduit dossier.
  6. Accédez à Administration -> Tâches.
  7. Sélectionnez le travail d' affectation en bloc et cliquez sur Exécuter.
  8. Vérifiez le statut de fin de tâche.
  9. Vérifiez la ou les vidéos assignées.
  10. Le fichier traité par le travail Bulk-Assigner sera déplacé vers le dossier BulkAssignProduct_Processed sous le répertoire Temp.

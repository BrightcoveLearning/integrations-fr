Préparer l'installation

Extraire les fichiers et dossiers et copiez-les dans le dossier de travail

  1. Créer un dossier [Magento_Root]/app/code/Brightcove/Upload
  2. Téléchargez le fichier archive (fichiers de code).
  3. Décompressez les fichiers/dossiers
  4. Drop/déplacer le contenu des fichiers décompressés vers le répertoire de l'étape 1

Activer les extensions et les plugins

Activez les éléments suivants :

  • php bin/magento module:enable Brigtcove_Upload
  • php bin/magento setup:upgrade
  • php bin/magento cache:clean
  • php bin/magento setup:static-content:deploy
  • php bin/magento cache:flush

Configurer les informations d'identification du module Brightcove

Pour compléter cette partie, vous devez obtenir les informations d'identification du client pour les API utilisées par l'extension. Pour plus d'informations sur la création d'informations d'identification, consultez Gestion des informations d'identification d'authentification API. Les autorisations minimales dont vous aurez besoin pour les informations d'identification sont indiquées ci-dessous :

  1. Accédez à Store -> Configuration -> Brightcove (onglet gauche) -> Configuration vidéo Brightcove
  2. Entrez les détails de la configuration de Brightcove :
    • ID client: Client REST/ID utilisateur
    • Secret client: REST Client Secret/mot de passe.
    • URL de base de l'API:
    • ID: ID de la boutique Brightcove
    • Catégorie Slot vidéo: Pour définir le nombre de slots vidéo sur la colonne de catégorie
    • Machine à sous vidéo principale: Pour définir le nombre de slots vidéo sur la colonne principale
    • Variante Slot vidéo : Pour définir le nombre de slots vidéo sur Variant Column
    • ID de courriel : Pour envoyer un e-mail lié à l'état du chargement groupé
  3. Une fois que tous les détails sont fournis correctement, puis Enregistrez la page de configuration. Les utilisateurs devraient désormais pouvoir accéder au module personnalisé Brightcove.

Désactiver extension

Si vous avez besoin de désactiver l'extension Brightcove, utilisez les commandes suivantes :

  1. php bin/magento module:disable Brightcove_Upload
  2. php bin/magento setup:upgrade
  3. php bin/magento cache:clean
  4. php bin/magento setup:static-content:deploy
  5. php bin/magento cache:flush

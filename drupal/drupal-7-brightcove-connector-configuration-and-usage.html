Connecteur Drupal 7-Brightcove : Configuration et utilisation

Introduction

Drupal Brightcove Video Connector vous permet de gérer les vidéos et les lecteurs Brightcove Video Cloud dans Drupal, et d'intégrer facilement des vidéos dans des pages Drupal. Cette page explique comment configurer le module Brightcove dans Drupal 7. Pour Drupal 8, voir Connecteur Drupal-Brightcove : Configuration.

L'installation Drupal est un processus compliqué qui se passe très facilement mal. Assurez-vous que votre contenu est sauvegardé avant de continuer.

Connectez-vous au compte Brightcove

  1. Pour connecter votre compte Brightcove, accédez à la page de l'API Brightcove en cliquant sur Configuration, puis faites défiler jusqu'au Paramètres de Brightcove option sous Médias. Vous accéderez à la page Paramètres de Brightcove. Pour ajouter un nouveau client API Brightcove, cliquez sur le Clients API onglet, puis sélectionnez Ajouter un client API. Cela vous amènera au formulaire à ajouter dans un client API.
  2. L'étiquette peut être n'importe quoi, mais utiliser le même nom que le nom du compte sur Brightcove facilitera l'identification.
  3. Pour rechercher l'ID de compte, regardez dans le coin supérieur gauche de votre tableau de bord Brightcove Studio.
  4. Pour obtenir l'ID client et le secret client, vous devrez enregistrer une application. Dans Brightcove Studio, accédez à Authentification et cliquez sur Ajouter une nouvelle application.
  5. Remplissez le nom et la brève description.
  6. Sélectionnez un compte. Sur la photo ci-dessous sont toutes les autorisations requises.
  7. Appuyez sur Enregistrer.
  8. L'ID client et le secret client s'affichent. Assurez-vous de copier et d'enregistrer Client Secret dans un emplacement sécurisé. Vous ne pourrez plus y accéder.
  9. Remplissez tous les ID et clés. Appuyez sur Enregistrer.
  10. Si vous rencontrez des problèmes, assurez-vous que la date et l'heure sont configurées, puis réessayez.

Ajout d'un champ vidéo Brightcove

Pour ajouter des vidéos Brightcove à vos publications Drupal, ajoutez d'abord un champ vidéo à la nouvelle page de contenu. Pour ce faire, accédez à « Structure » et cliquez sur « Types de contenu.«

Sélectionnez « Gérer les champs » dans le type de contenu auquel vous souhaitez ajouter des vidéos Brightcove. Ce processus peut être répété pour chaque type de contenu.

Cliquez sur « Ajouter un nouveau champ.« Étiquez-le, puis sélectionnez « Brightcove » pour Type de champ et « Vidéos (Upload/Parcourir) » pour le widget. Appuyez sur Enregistrer.

Votre champ vidéo devrait maintenant apparaître.

Publication et gestion des vidéos

Contrairement à Drupal 8, le connecteur Drupal 7 ne dispose pas d'un système de gestion de contenu distinct, de sorte que la plupart de la gestion du contenu doit être effectuée sur le site Brightcove Studio. Cependant, à partir de la nouvelle page de contenu, vous pouvez afficher toutes les images en appuyant sur Parcourir, ainsi que d'en télécharger de nouvelles.

Il existe deux façons d'attacher des vidéos existantes à une publication. Si vous connaissez le titre de la vidéo, la taper dans le champ Brightcove et permettre à la saisie semi-automatique de la trouver fonctionnera. Vous pouvez également cliquer sur le bouton Parcourir pour afficher et sélectionner toutes les vidéos.

Cliquer sur le bouton de téléchargement ouvrira le formulaire de téléchargement, vous permettant ainsi d'ajouter de nouvelles vidéos directement à partir de Drupal. Publiez le billet, et votre vidéo devrait apparaître.

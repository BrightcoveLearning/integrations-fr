Connecteur Drupal 8-Brightcove : Configuration

Introduction

Drupal Brightcove Video Connector vous permet de gérer les vidéos et les lecteurs Brightcove Video Cloud dans Drupal, et d'intégrer facilement des vidéos dans des pages Drupal. Notez que c'est pour Drupal 8. Pour Drupal 7, voir les instructions de Drupal 7.

L'installation Drupal est un processus compliqué qui se passe très facilement mal. Assurez-vous que votre contenu est sauvegardé avant de continuer.

Connectez-vous au compte Brightcove

  1. Pour connecter votre compte Brightcove, accédez à la page de l'API Brightcove en cliquant sur Configuration, puis faites défiler jusqu'au Compte API Brightcove option sous Médias. Cela vous amènera à la page client de l'API Brightcove. Pour ajouter un nouveau client API Brightcove, cliquez sur Ajouter le client API Brightcove.
  2. L'étiquette peut être n'importe quoi, mais utiliser le même nom que le nom du compte sur Brightcove facilitera l'identification.
  3. Pour obtenir l'ID client et le secret client, vous devrez enregistrer une application. Dans Brightcove Studio, accédez à Authentification et cliquez sur Ajouter une nouvelle application.
  4. Remplissez le nom et la brève description.
  5. Sélectionnez un compte. Sur la photo ci-dessous sont toutes les autorisations requises.
  6. Appuyez sur Enregistrer.
  7. L'ID client et le secret client s'affichent. Assurez-vous de copier et d'enregistrer Client Secret dans un emplacement sécurisé. Vous ne pourrez plus y accéder.
  8. Pour trouver l'ID de compte, accédez à la section Informations sur le compte des pages d'administration :
  9. Remplissez tous les identifiants et clés et sélectionnez Lecteur par défaut.
  10. Cliquez sur Enregistrer. Le client API doit maintenant apparaître dans la liste.

Synchronisation

Initialement, lorsque vous accédez au Nouveau contenu page et cliquez Vidéos de Brightcove, il n'y en aura pas, mais la possibilité d'ajouter une vidéo sera là. C'est parce que d'abord, vous devez synchroniser les données. Après la synchronisation initiale, le chron de Drupal garde tout à jour. Pour une synchronisation manuelle, cliquez sur le Rapports bouton en haut, puis cliquez sur Aperçu de l'état de Brightcove.

Sur le Aperçu de l'état de Brightcove page, appuyez sur Tout synchroniser.

Accédez à nouveau à la page vidéo Brightcove et vos vidéos doivent être disponibles.

Configuration des autorisations

Vous pouvez désormais afficher et publier des vidéos, mais des autorisations doivent être définies si des utilisateurs non administrateurs doivent interagir avec les vidéos. Pour configurer ces autorisations, accédez d'abord à la page des autorisations en cliquant sur Personnes puis Autorisations.

Faites défiler la page vers le bas pour trouver les autorisations Brightcove. Activez et désactivez en fonction de vos besoins. Les autorisations Brightcove sont automatiquement définies sur administrateur uniquement. Par conséquent, si vous souhaitez que d'autres personnes visualisent des vidéos, vous devez accorder ces autorisations.

Vous pouvez maintenant passer à la gestion du contenu ou à l' intégration de contenu.

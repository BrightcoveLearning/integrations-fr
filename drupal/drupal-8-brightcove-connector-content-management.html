Connecteur Drupal 8-Brightcove : Gestion de contenu

Introduction

Le connecteur Drupal-Brightcove vous permet de gérer les vidéos et les lecteurs Brightcove Video Cloud dans Drupal, et d'intégrer facilement des vidéos dans des pages Drupal. Notez que c'est pour Drupal 8. Pour Drupal 7, voir les instructions de Drupal 7.

Liste des vidéos

Pour voir toutes les vidéos Brightcove, cliquez sur Contenu, puis cliquez sur Vidéos de Brightcove.

Télécharger, modifier, supprimer

Pour télécharger, cliquez sur Ajouter une vidéo Brightcove. Pour modifier une vidéo, accédez à la vidéo dans la liste Brightcove et cliquez sur Modifier. Il y aura des options pour modifier les métadonnées. Pour supprimer, cliquez sur la flèche à côté de modifier, puis cliquez sur Supprimer.

Lors du téléchargement, il est important de définir le profil d'encodage sur celui que vous pouvez réellement accéder. Vérifiez votre compte studio Brightcove si vous n'êtes pas sûr.

Listes de lecture

Les listes de lecture peuvent être consultées en cliquant sur le Listes de lecture Brightcove languette. Pour ajouter une liste de lecture, cliquez sur le Ajouter Brightcove Playlist bouton.

Sur la page des nouvelles sélections, vous pouvez choisir si vous souhaitez créer une playlist manuelle ou une liste de lecture intelligente. Pour les sélections manuelles, vous pouvez ensuite sélectionner des vidéos. Pour les listes de lecture intelligentes, vous pouvez sélectionner les règles que vous souhaitez suivre.

Synchroniser avec Brightcove

Il existe un travail cron qui synchronise automatiquement les vidéos Brightcove, mais si quelque chose ne s'affiche pas, vous pouvez toujours essayer de les synchroniser manuellement.

  1. Pour commencer, cliquez sur Rapports, puis cliquez sur Aperçu de l'état de Brightcove.
  2. Puis clique Tout synchroniser au fond. Cela devrait synchroniser tout le contenu de Brightcove.

Assurez-vous de vérifier également l' intégration du contenu avec drupal !

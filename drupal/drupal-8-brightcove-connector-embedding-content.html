Connecteur Drupal 8-Brightcove : Intégration de contenu

Introduction

Le connecteur Drupal-Brightcove vous permet de gérer les vidéos et les lecteurs Brightcove Video Cloud dans Drupal, et d'intégrer facilement des vidéos dans des pages Drupal. Notez que c'est pour Drupal 8. Pour Drupal 7, voir les instructions de Drupal 7.

L'installation Drupal est un processus compliqué qui se passe très facilement mal. Assurez-vous que votre contenu est sauvegardé avant de continuer.

Remarque : ce document suppose que Brightcove a déjà été installé et configuré pour Drupal. Si vous ne l'avez pas déjà fait, revenez en arrière et installez et configurez l'extension Brightcove d'abord.

Bien que cette page se concentre sur la façon d'ajouter des vidéos Brightcove au contenu Drupal, les mêmes étapes peuvent être suivies pour ajouter des listes de lecture.

Mise en place

Naviguez vers le Se déployer rubrique et trouvez le Navigateur d'entités module. Cochez la case, puis cliquez sur Installer. Le moyen le plus simple de localiser le module est de commencer à taper le nom dans la barre de recherche. Installez également le Chaos Tools, Entity Browser IEF, Embed, Entity Embed, et CKEditor modules

Widget de sélection Brightcove

Configurez une vue personnalisée pour parcourir les vidéos. Clique sur le Structure onglet, puis cliquez sur Vues.

Cliquez sur Ajouter une vue. Cela vous mènera au Ajouter Vue page. Donnez un nom à la vue, puis sélectionnez Vidéo de Brightcove dans le menu Paramètres d'affichage. Cliquez sur Enregistrer et modifier.

Ensuite, cliquez sur le +Ajouter dans le Affiche rubrique et sélectionnez Navigateur d'entités. Puis frappe Enregistrer.

Pour le format, choisissez Table en cliquant sur le défaut option et en sélectionnant Table.

Maintenant, il est temps d'ajouter des champs pour le navigateur. Assurez-vous qu'il dit Navigateur d'entités au coin. Vous devez y être automatiquement basculé après l'avoir sélectionné.

Pour ajouter des champs, cliquez sur le Ajouter bouton à côté du Champs entête. Les champs peuvent être en grande partie une question de goût, mais assurez-vous d'avoir le Vidéo Brightcove : Formulaire de sélection groupée du navigateur d'entités en option afin que les vidéos puissent être sélectionnées. Le nom et la vignette sont également bons à avoir. Le seul domaine où il est très bon pas ont est celui qu'ils ont mis en place par défaut : le Entité rendue. L'entité rendue est la vidéo complète et toutes les méta-informations, et l'inclut dans le navigateur d'entités. Cela le rend encombrant et moins utile pour parcourir rapidement des vidéos.

N'oubliez pas de frapper Enregistrer.

Créer un navigateur d'entités

Pour configurer le navigateur d'entités, accédez à Configuration et sélectionnez le Navigateur d'entités option.

Sur la page résultante, cliquez sur Ajouter un navigateur d'entités au sommet. Sur la page résultante, ajoutez une étiquette pour votre nouveau widget, puis effectuez vos sélections. Sélectionner iFrame pour le plugin Display, Onglets pour le plugin de sélecteur de widget, et Pas d'affichage de sélection pour le plugin d'affichage de sélection.

Frapper suivant. Les pages suivantes ont soit des options par défaut fines, soit aucune option. Continuez jusqu'à ce que vous atteigniez l'écran final, qui a le mot Finir en bas au lieu du suivant. Sélectionner Vue dans le Ajouter un menu de plugin de widget, puis attribuez une étiquette à votre widget. Dans Afficher l'affichage, sélectionnez Navigateur vidéo Brightcove : Navigateur d'entité.

Ajouter un onglet Charger

Clique le Ajouter un plugin de widget menu déroulant à nouveau et sélectionnez Formulaire d'entité.

Cela ajoutera un formulaire sous le premier formulaire. Ajoutez une étiquette et sélectionnez Vidéo de Brightcove pour le Type d'entité et Empaqueter. Frappé Finir.

Bouton Ajouter Brightcove

Aller vers Configuration. Dans le Création de contenu rubrique, sélectionnez Boutons d'intégration de l'éditeur de texte. 

Cliquez sur Ajouter une intégration bouton. Donnez une étiquette à votre bouton, sélectionnez Entité pour le type encastré et Vidéo de Brightcove pour le type d'entité. Votre navigateur Entity doit être le navigateur que vous avez déjà créé pour le contenu Brightcove.

Si vous voulez une icône, vous devrez la télécharger ici, sinon il ne s'agit que de l'icône du bouton par défaut. Voici une petite icône de logo Brightcove que vous pouvez utiliser.

Revenir à Configuration et choisissez Formats de texte et éditeurs.

Choisissez de configurer le format de texte à l'endroit où vous souhaitez ajouter le bouton d'intégration de l'entité. Gardez à l'esprit que lors de la publication, vous devrez vous assurer d'utiliser ce format de texte. Vous pouvez répéter le processus pour ajouter le bouton à plusieurs formats.

S'assurer CKeditor est sélectionné pour le Éditeur de texte. Votre nouveau bouton Brightcove sera répertorié dans le Boutons disponibles, et peut être déplacé vers le Barre d'outils active.

S'assurer Afficher les entités incorporées est sélectionné et cliquez sur Enregistrer la configuration.

Ajouter une vue Brightcove

Pour que les vidéos s'affichent correctement, vous devez d'abord créer une nouvelle vue Brighcove. Pour cela, rendez-vous sur le Structure page et sélectionnez Modes d'affichage. Cliquez ensuite sur Modes d'affichage.

Cliquez sur Ajouter une nouvelle vue. Choisir Vidéo de Brightcove comme type d'entité du mode d'affichage. Donnez une étiquette à votre nouvelle vue et appuyez sur Enregistrer.

Vous devriez maintenant voir une vue répertoriée sous Brightcove Video.

Ensuite, activez le nouveau mode d'affichage. Aller à Paramètres vidéo Brightcove dans le Structure menu, puis sélectionnez le Onglet Gérer l'affichage. En bas, il y a un Paramètres d'affichage personnalisés boîte. Sélectionnez votre nouvelle vue personnalisée et appuyez sur Enregistrer.

Par défaut, la vue comporte beaucoup de métadonnées inutiles affichées avec la vidéo. À côté de la Par défaut lien d'affichage, cliquez sur le lien portant le nom de votre mode d'affichage. Désactivez toutes les métadonnées que vous ne souhaitez pas publier dans votre publication et activez toutes les métadonnées que vous souhaitez conserver. N'oublie pas de frapper Sauver.

Enfin, videz votre cache en accédant à Performances dans le Configuration menu et en cliquant Vider tous les caches.

Vous devriez maintenant pouvoir sélectionner Vidéo de Brightcove comme la vue à afficher !

Connecteur Drupal 7-Brightcove : Installation

Introduction

Le connecteur Drupal-Brightcove vous permet de gérer les vidéos et les lecteurs Brightcove Video Cloud dans Drupal, et d'intégrer facilement des vidéos dans des pages Drupal. Cet article va spécifiquement sur la façon d'installer le plug-in Drupal 7. Pour Drupal 8, veuillez consulter le guide d'installation de Drupal 8.

L'installation Drupal est un processus compliqué qui se passe très facilement mal. Assurez-vous que votre contenu est sauvegardé avant de continuer.

Étapes d'installation

Pour obtenir de meilleurs résultats, assurez-vous que vous avez déjà installé Composer. Trouvez des instructions sur la façon de procéder ici.

  1. Téléchargez et installez le php-API-wrapper. Déplacer vers sites/all/libraries/PHP-API-Wrapper
  2. Accédez à sites/all/libraries/PHP-API-Wrapper dans votre console. À partir du dossier, exécutez composer install.
  3. Maintenant, téléchargez le module Brightcove. Vous devrez également charger d'autres modules pour les dépendances :
    • Outils Chaos
    • API d'entité
    • API Bibliothèques
    • Date
    • Entité fichier
    • Médias
    • Vues
  4. Déplacez tous les modules vers le modules dossier.
  5. Allez sur votre site drupal. Accédez à la page Modules.
  6. Activer les modules suivants : ctools, api d'entité, bibliothèques, date, date api, popup de date, média, sources Internet multimédia, entité de fichier et vues. Tous ces éléments devraient être disponibles grâce à l'étape 3.
  7. Assurez-vous de définir la date et l'heure ! Votre module Brightcove ne fonctionnera pas si vous ne le faites pas.

Une fois que vous avez terminé, vous pouvez passer à Configuration.

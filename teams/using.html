Introduction

Avant d'utiliser Brightcove Media Manager, il doit être installé et configuré pour l'équipe. Voir Installation du connecteur pour les équipes pour les détails.

Après avoir ajouté le gestionnaire de médias à votre équipe comme expliqué dans la section précédente, vous pouvez ajouter et gérer des vidéos ajouter et gérer des listes de lecture et gérer des expériences à partir du gestionnaire de médias au sein des équipes.

Les modifications que vous apportez à partir du gestionnaire de médias dans Teams sont enregistrées sur votre compte Brightcove sur la plateforme Brightcove et vous pouvez les voir lorsque vous vous connectez via Brightcove Studio en ligne.

Ajouter des vidéos

  1. Ouvrez Teams et accédez à l'équipe et au canal où vous avez précédemment installé le gestionnaire de médias.
  2. Sélectionnez le Gestionnaire d'actifs onglet pour ouvrir le gestionnaire de médias.
  3. Clique le Ajouter des vidéos languette:
  4. Soit glisser déposer la vidéo que vous souhaitez télécharger sur le hot-spot indiqué, ou cliquez sur Parcourir les vidéos pour sélectionner la vidéo à ajouter :
  5. Une fois ajoutée, la vidéo est traitée sur la plate-forme de cloud vidéo Brightcove. Attendez la fin du traitement.
  6. La vidéo est prête à être éditée/gérée lorsque le traitement est terminé et vous pouvez cliquer sur le bouton Éditer vidéo lien comme indiqué ci-dessous :
  7. Vous serez dirigé vers la page de modification de la vidéo où vous pourrez ensuite modifier les métadonnées de la vidéo.

Gérer les vidéos

Dans le gestionnaire de médias, cliquez sur le Gérer les vidéos languette.

Modifier une vidéo

  1. Recherchez la vidéo que vous souhaitez modifier dans la liste des vidéos.
  2. Clique le Éditer relier:
  3. Modifiez les métadonnées de la vidéo comme vous le souhaitez :
  4. Lorsque vous avez terminé, faites défiler jusqu'en bas de la page d'édition et cliquez sur le bouton « Mettre à jour sur le cloud vidéo » pour enregistrer vos modifications dans le cloud vidéo Brightcove :

Obtenir le code d'intégration vidéo

  1. Trouvez la vidéo pour laquelle vous voulez le code d'intégration dans la liste des vidéos.
  2. Clique le Intégrer bouton:
  3. Dans le Intégrer la vidéo modale, cliquez sur le Copier dans le presse-papier lien pour copier le code d'intégration.
    Remarque : vous pouvez définir diverses configurations pour l'intégration afin de régir son comportement, et également le prévisualiser avant de copier le code, si vous le souhaitez.

Ajouter des listes de lecture

  1. Dans le gestionnaire de médias, cliquez sur le Gérer les listes de lecture languette.
  2. Clique le Ajouter une nouvelle liste de lecture bouton en bas de la liste des playlists :
  3. Saisissez un nom et une description, puis sélectionnez les vidéos que vous souhaitez inclure dans la liste de lecture.
  4. Clique le Créer bouton:
  5. La liste de lecture est traitée par le cloud Brightcove Video et s'affiche dans la liste des listes de lecture après le traitement.
    Remarque : le traitement peut prendre quelques instants et la playlist peut ne pas être immédiatement visible sur la liste des listes de lecture. Si vous le souhaitez, confirmez le traitement à l'aide de Brightcove. Studio.

Gérer les listes de lecture

Dans le gestionnaire de médias, cliquez sur le Gérer les listes de lecture languette.

Modifier une liste de lecture

  1. Recherchez la liste de lecture que vous souhaitez modifier.
  2. Clique le Éditer relier.
  3. Faites vos modifications et cliquez sur le sauvegarder bouton.

Obtenir le code d'intégration de la playlist

  1. Trouvez la liste de lecture pour laquelle vous souhaitez obtenir le code d'intégration.
  2. Clique le Intégrer bouton.
  3. Dans le Intégrer la liste de lecture modale, cliquez sur le Copier dans le presse-papier lien pour copier le code d'intégration.
    Remarque : vous pouvez définir diverses configurations pour l'intégration afin de régir son comportement, et également le prévisualiser avant de copier le code, si vous le souhaitez.

Gérer les expériences

Dans le gestionnaire de médias, cliquez sur le Gérer les expériences languette.

Aperçu de l'expérience

  1. Trouvez l'expérience que vous souhaitez prévisualiser.
  2. Clique le Aperçu relier.

Obtenez de l'expérience avec le code d'intégration

  1. Trouvez l'expérience pour laquelle vous souhaitez obtenir le code d'intégration.
  2. Clique le Intégrer bouton.
  3. Dans le Intégrer l'expérience modale, cliquez sur le Copier dans le presse-papier lien pour copier le code d'intégration.
    Remarque : vous pouvez définir diverses configurations pour l'intégration afin de régir son comportement, et également le prévisualiser avant de copier le code, si vous le souhaitez.

Ajouter une vidéo à un onglet de chaîne d'équipe

  1. Ouvrez Microsoft Teams et accédez à l'équipe sur laquelle vous souhaitez ajouter une vidéo.
  2. Ouvrez la chaîne sur laquelle vous souhaitez ajouter la vidéo.
  3. Clique le Ajouter un onglet menu:
  4. Sélectionnez l'application Brightcove Video Connector dans la liste des applications disponibles :
  5. Dans le modal de configuration, sélectionnez le compte Brightcove.
  6. Dans le modal de configuration, sélectionnez Incorporer la vidéo comme mode d'onglet Connecteur et cliquez sur Suivant:
  7. Sélectionnez la vidéo que vous souhaitez ajouter à l'onglet de chaîne, puis cliquez sur Enregistrer:
  8. Vous devriez maintenant voir la vidéo ajoutée dans l'onglet de la chaîne.

Ajouter une expérience à un onglet de canal d'équipe

  1. Ouvrez Microsoft Teams et accédez à l'équipe sur laquelle vous souhaitez ajouter une expérience.
  2. Ouvrez la chaîne sur laquelle vous souhaitez ajouter l'expérience.
  3. Clique le Ajouter un onglet menu:
  4. Sélectionnez l'application Brightcove Video Connector dans la liste des applications disponibles :
  5. Dans le modal de configuration, sélectionnez le compte Brightcove.
  6. Dans le modal de configuration, sélectionnez Embed Experience comme mode d'onglet Connector, puis cliquez sur Suivant:
  7. Sélectionnez l'expérience que vous souhaitez ajouter à l'onglet de chaîne, puis cliquez sur Enregistrer:
  8. Vous devriez maintenant voir l'expérience ajoutée dans l'onglet de la chaîne.

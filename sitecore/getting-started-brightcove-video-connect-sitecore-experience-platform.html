Installation de Brightcove Video Connect pour le Nuage vidéo Sitecore CMS

Cette rubrique traite du processus d'installation du module Brightcove Video Connect.

 

Installer Video Cloud Sitecore Media Framework 2.1

Brightcove Video Connect requiert Nuage vidéo Sitecore Media Framework doit être installé en premier. Les Nuage vidéo Sitecore Media Framework est installé à l'aide du Nuage vidéo Sitecore Assistant d'installation. Suivez les instructions affichées par l'assistant d'installation.

Relâcher les événements de page à partir

Les Nuage vidéo Sitecore Media Framework inclut un certain nombre d'événements de page qui doivent être libérés du workflow et publiés avant de pouvoir être utilisés.

  1. Ouvrez le Nuage vidéo Sitecore Boîte de travail
  2. Sélectionner le Workflow Analytics
  3. Déployez les événements de page suivants :
    • Lecture modifiée
    • Lecture terminée
    • Erreur de lecture
    • Lecture démarrée
  4. Publier les événements de page

Installez Brightcove Video Connect pour le Nuage vidéo Sitecore Plateforme d'expérience

Connexion vidéo Brightcove pour Nuage vidéo Sitecore est installé à l'aide du Nuage vidéo Sitecore Assistant d'installation. Suivez les instructions affichées par l'assistant d'installation. Les Nuage vidéo Sitecore plug-in est disponible en téléchargement via un lien dans le tableau ci-dessus. Ce fichier zip inclut à la fois la source et le package de plug-in.  Décompressez-le d'abord, puis téléchargez le package zip interne sur votre Nuage vidéo Sitecore exemple.

Configuration de la base de données

Afin que le Nuage vidéo Sitecore Media Framework rapporte pour fonctionner, une nouvelle table de faits doit être ajoutée à la base de données de rapports Experience Database (xDB). Un script qui va créer cette nouvelle table est disponible sur SDN. Vous devez exécuter ce script sur votre base de données de rapports xDB. Cette étape doit être effectuée par votre administrateur de base de données.

Note : Il est possible d'installer plusieurs Nuage vidéo Sitecore Connecteurs Media Framework. Ce script ne doit être exécuté qu'une seule fois par base de données de rapports, quel que soit le nombre de connecteurs que vous avez installés.

Configuration multi-serveur

Nuage vidéo Sitecore Media Framework peut être installé dans un environnement de création de contenu multiserveur. Si votre Nuage vidéo Sitecore environnement est configuré de cette manière, vous devez configurer les éléments suivants :

Serveurs Gestion de contenu

Si vous utilisez le planificateur pour synchroniser les médias, un seul serveur de gestion de contenu (CM) doit être configuré. Supprimez le travail planifié des fichiers de configuration des autres serveurs CM. Les Nuage vidéo Sitecore la file d'attente d'événements gérera la synchronisation des autres serveurs CM.

Note : La configuration du planificateur sur plusieurs serveurs CM ne pose pas de problème, mais il n'y a aucun avantage à le faire.

Serveurs de diffusion de

Sur chaque serveur de remise de contenu (CD), supprimez l'index suivant du fichier de configuration spécifié. Cet indice n'est nécessaire que dans l'environnement CM.

Dossier : Video Cloud Sitecore .MediaFramework.Services.Brightcove.config
Nom de l'index : MediaFramework_Brightcove_Index
 

Lors de la configuration des serveurs de CD, il est facile de copier des fichiers de configuration à partir d'un serveur CM. Si vous effectuez cette opération et que vous avez ajouté un planificateur pour synchroniser les médias, vous devez supprimer le travail planifié des fichiers de configuration sur les serveurs de CD. La synchronisation des médias ne doit pas s'exécuter sur les serveurs de CD. Il devrait fonctionner sur un serveur CM. La publication permet de rendre les éléments multimédias disponibles sur les serveurs de CD.

Mise en relation Nuage vidéo Comptes

Cette rubrique explique comment créer un compte dans Nuage vidéo Sitecore qui correspond à un Brightcove Nuage vidéo Compte.

 

Obtenir les informations d'identification de compte de Brightcove

Pour que Media Framework puisse se connecter à Brightcove Video Cloud, les informations d'identification du compte de Brightcove doivent être ajoutées à Video Cloud Sitecore. . Cette section décrit comment localiser les informations d'identification du compte dans Brightcove. La section suivante explique comment ajouter ces informations d'identification à Video Cloud Sitecore.

  1. Connectez-vous à Brightcove Video Cloud. Notez la valeur de l'ID de l'éditeur. Vous devrez le saisir dans Nuage vidéo Sitecore dans la section suivante.
  2. Accédez à Admin.
  3. Accédez à l'authentification API.
  4. Enregistrez une nouvelle demande.
  5. Nommez l'application, définissez l'accès au compte et activez les API.
  6. Notez l'ID client et le secret client. Vous devrez les saisir dans Nuage vidéo Sitecore dans la section suivante.

Ajouter un compte Brightcove à Nuage vidéo Sitecore

Cette section explique comment ajouter un compte Brightcove à Video Cloud Sitecore.

  1. Connectez-vous au Nuage vidéo Sitecore Bureau.
  2. Ouvrez la médiathèque.
  3. Accédez à Médiathèque > Media Framework > Comptes.
  4. Ajoutez un nouvel élément Compte Brightcove.
  5. Entrez l'ID éditeur, l'ID client et les valeurs Secret client indiquées dans la section précédente.
  6. Enregistrez l'élément.

Définir Nuage vidéo Joueurs

Un lecteur est nécessaire pour afficher une seule vidéo ou une playlist de vidéos. Un lecteur définit le type de capacités disponibles pour le visiteur qui veut regarder une vidéo. Nuage vidéo Sitecore Media Framework permet aux auteurs de contenu d'ajouter de nouveaux lecteurs, de modifier des lecteurs existants et de supprimer des lecteurs existants.

 

Ajouter un lecteur

Cette section décrit comment créer manuellement le Nuage vidéo Sitecore élément qui correspond à un joueur.

  1. Dans Brightcove Video Cloud, accédez à la section Lecteurs.
  2. Sélectionnez le lecteur que vous souhaitez créer dans Video Cloud Sitecore. Notez la valeur de l'ID du joueur. Vous devrez le saisir dans Nuage vidéo Sitecore dans la section suivante.
  3. Dans Video Cloud Sitecore, dans la médiathèque, sélectionnez l'élément Compte Brightcove.
  4. Sélectionnez l'élément Comptes.
  5. Sélectionnez l'élément Joueurs.
  6. Ajoutez un nouvel élément Brightcove Video Player. Vous serez invité à entrer l'ID du joueur. Entrez la valeur de Brightcove Video Cloud.

Définir les joueurs par défaut

Pour que les vidéos et les listes de lecture soient prévisualisées dans Video Cloud Sitecore, les lecteurs par défaut doivent être spécifiés. Cette section décrit comment définir les joueurs par défaut.

  1. Dans Video Cloud Sitecore, dans la médiathèque, sélectionnez l'élément Compte Brightcove.
  2. Sélectionnez l'élément Comptes.
  3. Sélectionnez l'élément Paramètres.
  4. Définissez les valeurs du champ Lecteur vidéo par défaut et du champ Lecteur de liste de lecture par défaut.
  5. Enregistrer l'élément

Modifier le joueur

Les éléments de joueur sont édités comme n'importe quel autre Nuage vidéo Sitecore éléments.

Supprimer le lecteur

Les éléments du joueur sont supprimés comme n'importe quel autre Nuage vidéo Sitecore éléments.

Note : La suppression d'un lecteur de Video Cloud Sitecore n' entraînera pas la suppression du lecteur de Brightcove Video Cloud.

Importation, ajout, prévisualisation et suppression de vidéos

Nuage vidéo Sitecore Media Framework permet à un auteur de contenu de télécharger des vidéos depuis sa machine locale, ainsi que de modifier et de supprimer des vidéos existantes. De nouvelles balises peuvent être définies pour organiser le contenu vidéo.

 

Importation de vidéos

UNE Nuage vidéo Sitecore l'auteur ou l'administrateur de contenu peut déclencher manuellement le processus d'importation des médias.

Lorsque vous importez des vidéos, Video Cloud Sitecore reflétera le jour actuel où les vidéos ont été importées plutôt que la date de téléchargement à l'intérieur de Cloud vidéo. Si vous déplacez une implémentation Video Cloud Sitecore déjà existante, vous pouvez migrer le contenu précédent et le connecteur reconnaîtra ces vidéos.

Le plug-in Brightcove Video Connect inclut un agent qui synchronise automatiquement les médias régulièrement. L'agent est défini à l'emplacement suivant :

Fichier de configuration : Video Cloud Sitecore .MediaFramework.Services.Brightcove.config
Nom de l'agent : MediaFramework_Import_Brightcove
  1. Affichez le ruban Media Framework en cliquant avec le bouton droit sur le menu supérieur et en sélectionnant Media Framework.
  2. Cliquez sur le ruban Media Framework.
  3. Cliquez sur le bouton Importer tout.
  4. Au fur et à mesure que le processus d'importation s'exécute, une zone d'état apparaît. Lorsque la zone d'état disparaît, le processus d'importation est terminé.
  5. Pour confirmer que le média a été importé, cliquez sur l'onglet Rechercher.
  6. Effectuez une recherche.
  7. Filtrer les vidéos.
  8. Les résultats de la recherche sont affichés.

Ajouter une vidéo

Nuage vidéo Sitecore Media Framework permet aux auteurs de contenu d'ajouter de nouvelles vidéos.

  1. Ouvrez le Nuage vidéo Sitecore Médiathèque.
  2. Accédez au compte contenant les vidéos pour laquelle vous souhaitez créer une liste de lecture.
  3. Accédez à l'élément Contenu multimédia.
  4. Cliquez sur Brightcove Video.
  5. Cliquez sur Ajouter des fichiers.
  6. Sélectionnez le fichier que vous souhaitez télécharger depuis votre ordinateur local.
  7. Cliquez sur Démarrer.
  8. Une fois la vidéo téléchargée, fermez la fenêtre de téléchargement.

Définir des tags

Nuage vidéo Sitecore Media Framework permet aux auteurs de contenu d'ajouter de nouvelles balises.

  1. Ouvrez le Nuage vidéo Sitecore Médiathèque.
  2. Accédez au compte contenant les vidéos pour lesquelles vous souhaitez créer une balise.
  3. Accédez à l'élément Balises.
  4. Cliquez sur Brightcove Tag.
  5. Saisissez un nom d'article. Cela deviendra la balise.

Modifier la vidéo et les balises

Les éléments vidéo et tag sont édités comme n'importe quel autre Nuage vidéo Sitecore éléments.

Supprimer la vidéo et les balises

Les éléments vidéo et tag sont supprimés comme n'importe quel autre Nuage vidéo Sitecore éléments.

Rechercher des vidéos et des listes de lecture

Les vidéos et les listes de lecture peuvent être trouvées en recherchant des chaînes de texte dans leur nom et leur description. Les vidéos peuvent également être trouvées en recherchant des chaînes de texte dans leurs champs personnalisés.

 

Recherche

Cliquez sur l'onglet Rechercher pour rechercher des médias. Saisissez le texte à rechercher dans les champs Nom ou Description.

Cliquez sur la loupe pour effectuer une recherche.

Filtrage

Les résultats de la recherche peuvent être filtrés par type, comme la vidéo ou la liste de lecture, par date et par d'autres attributs.

Les résultats de la recherche sont affichés.

Importation, ajout, prévisualisation et suppression de listes de lecture

Une playlist est une collection de vidéos dans un seul lecteur. Nuage vidéo Sitecore Media Framework permet aux auteurs de contenu d'ajouter de nouvelles listes de lecture, de modifier des listes de lecture existantes et de supprimer des listes de lecture existantes.

Lorsque vous importez des listes de lecture, Video Cloud Sitecore reflète le jour actuel où les listes de lecture ont été importées plutôt que la date de création à l'intérieur de Cloud vidéo. Si vous déplacez une implémentation Video Cloud Sitecore déjà existante, vous pouvez migrer le contenu précédent et le connecteur reconnaîtra ces vidéos.
 

Ajouter une playlist

Nuage vidéo Sitecore Media Framework permet aux auteurs de contenu d'ajouter de nouvelles listes de lecture.

  1. Ouvrez le Nuage vidéo Sitecore Médiathèque.
  2. Accédez au compte contenant les vidéos pour laquelle vous souhaitez créer des sélections.
  3. Accédez à l'élément Contenu multimédia.
  4. Ajoutez un nouvel élément de la playlist Brightcove.
  5. Dans le champ Nom, entrez un nom descriptif pour la vidéo.
  6. Pour le champ ID vidéo, sélectionnez les vidéos que vous souhaitez inclure dans la liste de lecture.

Modifier la playlist

Les éléments de la liste de lecture sont édités comme n'importe quel autre Nuage vidéo Sitecore éléments.

Supprimer la playlist

Les éléments de la liste de lecture sont supprimés comme n'importe quel autre Nuage vidéo Sitecore éléments.

Ajouter une vidéo à Nuage vidéo Sitecore Site Internet

Nuage vidéo Sitecore Media Framework permet aux auteurs de contenu d'insérer des médias dans leurs pages de diverses manières. Ce chapitre décrit les options disponibles.

 

Éditeur de texte riche

Les champs de texte enrichi permettent aux auteurs de contenu d'insérer du contenu formaté dans leurs pages. Les auteurs de contenu utilisent l'éditeur de texte enrichi pour créer et modifier ce contenu formaté. Cette section couvre les fonctionnalités qui Nuage vidéo Sitecore Media Framework s'ajoute à l'éditeur de texte enrichi.

Intégrer les médias

Les médias peuvent être incorporés directement dans un champ de texte enrichi.

  1. Dans l'éditeur de texte enrichi, cliquez sur le bouton Incorporer un média.
  2. Utilisez l'écran de recherche pour trouver le média que vous souhaitez insérer. Cliquez sur la ligne qui représente le média à insérer. Cliquez sur le bouton Suivant >.
  3. Selon le type de média que vous sélectionnez, vous verrez des options supplémentaires. Par exemple, si vous sélectionnez une vidéo, vous devez spécifier le lecteur qui doit être utilisé pour afficher la vidéo, ainsi que les dimensions du lecteur. Cliquez sur Finish (Terminer).
  4. Le média est intégré dans le champ de texte enrichi.

Intégrer le lien média

Un lien vers un média peut être inséré dans un champ de texte enrichi.

  1. Dans l'éditeur de texte enrichi, cliquez sur le bouton Incorporer un lien multimédia.
  2. Utilisez l'écran de recherche pour trouver le média que vous souhaitez insérer. Cliquez sur la ligne qui représente le média à insérer. Cliquez sur le bouton Suivant >.
  3. Entrez une valeur pour le titre du lien. Il s'agit du texte qui apparaîtra pour le lien média.
    En outre, selon le type de média que vous sélectionnez, vous verrez des options supplémentaires. Par exemple, si vous sélectionnez une vidéo, vous devez spécifier le lecteur qui doit être utilisé pour afficher la vidéo, ainsi que les dimensions du lecteur. Cliquez sur Finish (Terminer).
  4. Un lien est incorporé dans le champ de texte enrichi.

Rendu

Les rendus permettent de positionner le contenu à un emplacement spécifique sur une page. Nuage vidéo Sitecore Media Framework inclut un rendu qui permet aux auteurs de contenu d'insérer des médias dans les pages.

Intégrer le rendu des médias

  1. Utilisez l'Éditeur de page pour ajouter un rendu à un espace réservé.
  2. Sélectionnez le rendu Incorporer le média.
  3. Le rendu a été ajouté à l'espace réservé. Cliquez maintenant sur le bouton Incorporer un élément multimédia.
  4. Utilisez l'écran de recherche pour trouver le média que vous souhaitez insérer. Cliquez sur la ligne qui représente le média à insérer. Cliquez sur le bouton Suivant >.
  5. Selon le type de média que vous sélectionnez, vous verrez des options supplémentaires. Par exemple, si vous sélectionnez une vidéo, vous devez spécifier le lecteur qui doit être utilisé pour afficher la vidéo, ainsi que les dimensions du lecteur. Cliquez sur Finish (Terminer).
  6. Le média a été sélectionné pour le rendu.

Synchronisation entre Nuage vidéo Sitecore et Nuage vidéo

Cette section traite de la synchronisation des vidéos et des listes de lecture entre Nuage vidéo Sitecore pour les scénarios suivants :

  • Lorsque des vidéos et des listes de lecture sont supprimées de Brightcove
  • Lorsque des vidéos et des listes de lecture sont ajoutées Nuage vidéo Sitecore
  • Lorsque des vidéos et des listes de lecture sont supprimées de Nuage vidéo Sitecore

Importation de Brightcove vers Nuage vidéo Sitecore était couvert auparavant ici.

Exportation de vidéo

Dans Video Cloud Sitecore, les médias Brightcove sont accessibles à l'aide de Video Cloud Sitecore éléments créés pendant le processus d'importation de médias. Puisqu'ils sont Nuage vidéo Sitecore éléments, il est possible pour les utilisateurs de modifier ces éléments. Le processus d'exportation multimédia est responsable de l'exportation des modifications à partir de Nuage vidéo Sitecore à Brightcove.

Exportation automatique des médias

Lorsqu'un Nuage vidéo Sitecore l'élément correspondant au support Brightcove est créé, enregistré ou supprimé dans le Nuage vidéo Sitecore client, ces modifications seront transmises à Brightcove.

Exportations en temps réel

Lorsqu'un nouvel élément multimédia est créé dans Video Cloud Sitecore, une exportation en temps réel est effectuée par défaut. Cela signifie que la tâche d'exportation est effectuée immédiatement.

Empêcher l'exportation automatique

Vous ne voudrez peut-être pas que des modifications soient apportées dans Nuage vidéo Sitecore à exporter vers Brightcove. Vous voudrez peut-être la connexion de Nuage vidéo Sitecore vers Brightcove pour être en lecture seule, où les modifications Brightcove sont importées vers Nuage vidéo Sitecore mais où Nuage vidéo Sitecore les modifications ne sont pas exportées vers Brightcove.

Afin d'éviter les modifications apportées à Nuage vidéo Sitecore d'être exporté vers Brightcove, effacez la valeur du paramètre suivant :

Fichier de configuration : Video Cloud Sitecore .MediaFramework.config
Nom du paramètre : Video Cloud Sitecore .MediaFramework.ExportDatabases
 

Note : Ce paramètre s'applique à la Nuage vidéo Sitecore Media Framework, pas seulement le plug-in Brightcove Video Connect. La modification de ce paramètre s'appliquera à tous les connecteurs et à tous les comptes.

Nettoyage des données

Les processus de nettoyage sont utilisés pour garder Nuage vidéo Sitecore et les données Brightcove synchronisées.

Nuage vidéo Sitecore Nettoyer

Nuage vidéo Sitecore le nettoyage est le processus qui s'exécute lorsque le média est supprimé dans Brightcove et le Nuage vidéo Sitecore les éléments doivent être supprimés. Puisque Brightcove ne connaît rien de Video Cloud Sitecore, Brightcove ne peut pas notifier Video Cloud Sitecore ce média a été supprimé. Par conséquent, Nuage vidéo Sitecore doit être proactif.

Nettoyage manuel

UNE Nuage vidéo Sitecore L'auteur ou l'administrateur de contenu peut déclencher manuellement le processus de nettoyage en cliquant sur le bouton suivant dans l'éditeur de contenu :

Nettoyage automatique

Le plug-in Brightcove Video Connect inclut un agent qui exécute automatiquement le processus de nettoyage régulièrement. L'agent est défini à l'emplacement suivant :

Fichier de configuration : Video Cloud Sitecore .MediaFramework.Services.Brightcove.config
Nom de l'agent : MediaFramework_Cleanup_Brightcove

Nettoyage Brightcove

Le nettoyage Brightcove est le processus qui s'exécute lorsqu'un élément multimédia est supprimé dans Nuage vidéo Sitecore et le média correspondant dans Brightcove doit être supprimé ou désactivé. Étant donné que Brightcove sait que l'élément multimédia a été supprimé, Nuage vidéo Sitecore est en mesure d'informer Brightcove que le média correspondant dans Brightcove doit être supprimé ou désactivé.

Lorsqu'un élément multimédia est supprimé dans le Nuage vidéo Sitecore client, une entrée est effectuée dans la file d'attente d'exportation. Le processus est le même que lorsqu'un élément multimédia est enregistré.

Analyses d'événements de lecture

Les Nuage vidéo Sitecore Media Framework permet d'enregistrer des événements de lecture vidéo pour une utilisation dans le Nuage vidéo Sitecore Système de marketing numérique (DMS). Cette section couvre les options disponibles.

Événements de lecture

Les événements de lecture sont utilisés pour enregistrer les événements de page DMS à la suite d'actions du lecteur vidéo. Par exemple, lorsque la lecture vidéo démarre, un événement de page DMS peut être écrit.

Événements de lecture par défaut

Nuage vidéo Sitecore Media Framework est fourni avec les événements de lecture suivants :

  • Lecture démarrée
  • Lecture terminée
  • Erreur de lecture

Configurer les événements de lecture pour le compte

Par défaut, les événements de lecture démarrés et terminés de lecture sont enregistrés. Cette option est activée sur l'élément Paramètres sous l'élément Compte Brightcove.

L'élément Paramètres comporte un champ nommé Règles d'événements de lecture. Une règle est définie qui indique à Media Framework d'enregistrer les événements spécifiés dans les vidéos et les sélections.

Désactiver les événements de lecture pour le compte

Tous les événements de lecture peuvent être désactivés pour un compte spécifique.

  1. Dans l'Éditeur de contenu, accédez au compte sous Médiathèque > Media Framework > Comptes.
  2. Sélectionnez l'élément Paramètres.
  3. Le champ Règles d'événements de lecture décrit les conditions qui doivent être remplies pour que les événements de lecture soient enregistrés. Par défaut, la condition est « where true », ce qui signifie « toujours ». Cliquez sur le bouton Modifier la règle pour modifier la condition ou pour supprimer la règle existante.

Spécifier les événements de lecture pour les médias

Chaque playlist et élément vidéo comporte un champ nommé Événements. Ce champ détermine les événements de lecture qui seront gérés pour l'élément spécifié.

La capture d'écran suivante indique que des événements de lecture ont été spécifiés pour la vidéo :

Note : Les valeurs standard peuvent être utilisées pour s'assurer que certains événements sont enregistrés pour toutes les playlists et vidéos, mais le champ permet également aux playlists et vidéos individuelles d'avoir leurs propres événements sélectionnés.

Remarque : Ces événements ne seront enregistrés que si l'élément Paramètres sous l'élément de compte Brightcove a son champ Règles d'événements de lecture configuré avec l'action « Ajouter des événements d'élément multimédia ». Voir point 3.1.2.

Créer de nouveaux événements de lecture

Vous pouvez ajouter de nouveaux événements de lecture afin de capturer des informations plus détaillées sur la lecture vidéo.

  1. Dans Éditeur de contenu, accédez à Système > Modules > Media Framework > Evénements de lecture
  2. Ajouter un nouvel élément à l'aide du modèle d'événement de lecture
  3. Pour le champ Événement de page, sélectionnez Lecture modifiée
  4. Pour le champ Paramètre, vous pouvez saisir une valeur dans l'un des formats suivants :
    1. Nombre - une fois que la vidéo est lue ce nombre de secondes, l'événement de page est enregistré. Un exemple de valeur valide est 15
    2. Pourcentage - une fois que la vidéo est lue ce montant, l'événement de page est enregistré. Un exemple de valeur valide est 50%
  5. Enregistrer l'élément
  6. Publier l'élément
  7. Maintenant, vous pouvez ajouter l'événement de nouvelle page à votre playlist et vidéos

Remarque : N'oubliez pas d'attribuer le nouvel événement de lecture à votre playlist et à vos vidéos. Sinon, l'événement ne sera pas enregistré.

Rapports

Les Nuage vidéo Sitecore Media Framework inclut un certain nombre de rapports relatifs à la lecture vidéo.

Lecture commencée

Ce rapport affiche la liste des vidéos qui ont déclenché l'événement de démarrage de la vidéo.

Lecture terminée

Ce rapport affiche la liste des vidéos qui ont déclenché l'événement terminé.

Lecture incomplète

Ce rapport affiche la liste des vidéos qui ont déclenché l'événement de démarrage de la vidéo, mais pas l'événement terminé par la vidéo.

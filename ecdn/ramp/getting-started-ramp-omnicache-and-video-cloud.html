Introduction

Rampe AltitudeCDN ™ OmniCache Le produit est un système de distribution vidéo d'entreprise qui utilise la technologie de cache proxy pour réduire le trafic vidéo sur la passerelle Internet ou les liaisons à bande passante limitée au sein du réseau d'entreprise. Avec OmniCache, les ordinateurs de bureau clients et les périphériques mobiles utilisant Brightcove Player retirent le flux directement depuis le serveur OmniCache, puis le serveur OmniCache extrait le flux du CDN Brightcove, s'il ne dispose pas déjà d'une copie locale dans le cache. OmniCache est une solution entièrement sur site fournie en tant qu'appliance logicielle et est gérée par la console de gestion AltitudecDN™ Altimètre.  Omnicache peut être utilisé de manière autonome ou de concert avec AltitudecDN™ Multicast+ pour les réseaux mixtes unicast/multicast.

L'intégration avec Brightcove utilise le plug-in OmniCache Video.js pour réécrire l'URL du flux sur le serveur OmniCache.  Pour plus de détails sur la configuration du serveur OmniCache pour une utilisation avec Video Cloud, consultez cette TechNote de Ramp.

Avant de suivre les instructions ci-dessous, veuillez compléter le processus de démarrage rapide OmniCache avec l'équipe Ramp pour vous assurer que le serveur OmniCache et le réseau local sont correctement configurés.

Configuration du lecteur

Commencez par configurer un lecteur Brightcove pour utiliser le plug-in OmniCache. Créez un nouveau joueur avec l'habillage et les options souhaitées.

Ensuite, ajoutez le plug-in OmniCache Video.js.

Copiez et collez à partir d'ici :

    https://livetools.ramp.com/omnicache/plugin/brightcove-omnicache-v1.4.js

Enfin, définissez les options du plug-in OmniCache. Remplacez la valeur proxy « omnicachedemo.ramp.com » par le nom DNS de votre serveur OmniCache. Vous pouvez également spécifier un équilibreur de charge DNS pour répartir le trafic vidéo sur plusieurs serveurs OmniCache.

Copiez et collez à partir d'ici :

    {
        "proxy": "omnicachedemo.ramp.com",
        "verbose": true,
        "timeout": 2000
    }

Intégrer Brightcove Player

Enfin, intégrez le lecteur Brightcove à votre page Web en utilisant le processus de publication standard du lecteur.

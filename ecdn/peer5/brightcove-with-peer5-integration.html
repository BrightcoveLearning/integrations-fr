Étapes pour ajouter le plugin Peer5 à un lecteur

  1. Accédez au module Players de Studio et sélectionnez votre lecteur (ou dupliquez votre lecteur existant pour créer un nouveau lecteur compatible Peer5).
  2. Lorsque le lecteur a été créé, cliquez sur le nom du joueur pour le modifier.
  3. Dans la barre latérale, accédez au Plugins section.
  4. Choisissez Scripts dans le menu à onglets.
  5. Cliquez sur Ajouter un script.
  6. Collez la chaîne du plugin ci-dessous dans la boîte contextuelle (récupérez l'ID de compte depuis votre compte Peer5). 
    https://api.peer5.com/peer5.brightcove.plugin.loader.js?id={Account_ID}
  7. Cliquez sur Save.
  8. Cliquez sur Publier et intégrer, puis sur Publier les modifications:

Tester l'intégration

Étapes à suivre pour tester l'intégration

  1. Accédez au module Media dans Studio.
  2. Sélectionnez une vidéo, cliquez sur Publier et intégrer, puis choisissez Web Player:
  3. Sélectionnez votre lecteur compatible Peer5.
  4. Cliquez sur le lien ci-dessous « Aperçu dans le navigateur » pour ouvrir le lecteur dans un nouvel onglet.
  5. Ajouter &peer5_show_stats=true à la fin de l'URL.

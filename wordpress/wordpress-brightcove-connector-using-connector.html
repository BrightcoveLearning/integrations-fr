Connecteur WordPress-Brightcove : Utilisation du connecteur

Cette rubrique explique l'utilisation du connecteur Wordpress-Brightcove pour gérer et publier des vidéos dans Wordpress.

Introduction

Le connecteur WordPress-BrightCove vous permet de gérer les vidéos et les lecteurs Brightcove Video Cloud dans WordPress, et d'intégrer facilement des vidéos dans des pages WordPress.

Publication de lecteur vidéo

La publication de lecteur vidéo permet à l'utilisateur d'insérer facilement des vidéos de son compte Brightcove sur son site WordPress.

Pour publier une vidéo, commencez par naviguer simplement vers la nouvelle page de publication. Au niveau des fonctionnalités principales, il y a un bouton « Ajouter un média » sous le titre et au-dessus de l'éditeur. Avec Brightcove Video Connect, il y a aussi un bouton « Brightcove Media ». Cliquez dessus pour ouvrir le modal média Brightcove.

À partir de là, l'utilisateur peut sélectionner l'image qu'il souhaite utiliser en cliquant dessus. La vignette de la vidéo apparaîtra sur le côté, avec les options. Pour trouver une vidéo spécifique, vous pouvez effectuer une recherche par titre, balise ou description.

L'utilisateur peut alors choisir le lecteur qu'il souhaite utiliser et personnaliser des options telles que la lecture automatique, le style d'intégration et le dimensionnement.

Il y a deux options pour insérer la vidéo. Il suffit de cliquer sur le bouton « Insérer dans la publication » insère la vidéo dans le post, mais l'utilisateur peut également copier le shortcode et le coller dans le post. Si vous utilisez le bouton « Insérer dans la publication », le code court sera toujours visible dans la vue texte brut.

La vidéo apparaîtra sur la nouvelle page de publication, mais pour voir à quoi elle ressemblera publiée, vérifiez l'aperçu du post. Cela peut être très différent.

Publication de sélections

Les playlists peuvent être publiées exactement de la même manière que les vidéos individuelles. Notez qu'il doit y avoir un lecteur Brightcove compatible avec la liste de lecture pour pouvoir publier des listes de lecture.

Gestion des contenus vidéo

L'utilisateur peut télécharger des vidéos sur Brightcove et mettre à jour les métadonnées directement à partir de son tableau de bord WordPress.

Téléchargement vidéo

Pour télécharger une vidéo sur Brightcove, à partir de la page d'administration de Brightcove sur WordPress, cliquez sur « Ajouter un nouveau ».«

À partir de là, vous pouvez sélectionner un fichier ou faire glisser un fichier vers la page à télécharger. Vous pouvez sélectionner plusieurs fichiers à télécharger en une seule fois

Le fichier apparaîtra sur la page. Pour ajouter des métadonnées, cliquez sur le nom du fichier et l'option s'ouvrira.

Lorsque vous êtes prêt, appuyez sur Démarrer le téléchargement.

Les vidéos peuvent être téléchargées de la même manière à partir du modal média sur la nouvelle page de publication en cliquant sur le lien « Télécharger ».

Échec

Si la vidéo ne parvient pas à télécharger, cela peut poser un problème avec les autorisations de fichier. Vérifiez que le serveur Web dispose des autorisations en lecture/écriture pour le dossier wp-content/uploads. La vidéo sera temporairement téléchargée ici.

Brightcove n'ingère pas non plus derrière un pare-feu. Si c'est le problème, la nouvelle vidéo sera générée, mais pas transcodée.

Pour contourner le problème, accédez à votre compte Brightcove et remplacez le fichier source. Le pare-feu doit être ouvert pour que Brightcove puisse ingérer des vidéos de WordPress.

N'oubliez pas de décocher la case pour conserver les vignettes et les affiches existantes. Sinon, il restera une boîte grise.

Une fois le téléchargement terminé, votre vidéo devrait être disponible dans WordPress.

Pour plus d'informations, consultez les problèmes de connexion réseau et API de débogage avec les intégrations CMS et DAM.

Métadonnées

À partir de WordPress, un utilisateur peut mettre à jour le nom, la description courte, la description longue et les balises d'une vidéo.

De nouvelles vignettes, affiches et pistes de légende peuvent également être téléchargées, mais comme pour le téléchargement de vidéos, vous pouvez être bloqué par des pare-feu ou des autorisations.

À partir de la version 1.7.0, vous pourrez également sélectionner un dossier Video Cloud prédéfini pour une vidéo. Cette action déplacera la vidéo vers ce dossier dans Wordpress et Video Cloud. Remarque : Vous devez créer des dossiers dans Video Cloud.

D'autres champs de métadonnées, y compris les champs personnalisés, doivent être modifiés dans Brightcove Studio.

Suppression

Les utilisateurs peuvent supprimer des vidéos sur la page d'édition. Un avertissement apparaîtra pour confirmer.

Gestion de la playlist

Les playlists ne peuvent pas être créées à partir de WordPress, mais une fois qu'elles existent, elles peuvent être modifiées. Les vidéos peuvent être ajoutées ou supprimées, ou déplacées vers le haut ou vers le bas dans la liste.

Joueurs

Les lecteurs peuvent également être créés ou modifiés uniquement dans Brightcove Studio, mais certains aspects, tels que la taille, peuvent être modifiés pour la vidéo spécifique lors de sa publication. Veuillez noter qu'un lecteur de playlist doit déjà exister avant de pouvoir publier des listes de lecture.

Expérience dans la page

Le plugin Brightcove prend en charge les expériences en page. Bien que l'option soit visible pour tout le monde, l'option doit être disponible pour le compte spécifique, sinon elle ne fonctionnera pas. Seuls les comptes qui ont accès à l'expérience dans la page peuvent l'utiliser dans Wordpress.

Il y a deux façons d'intégrer des expériences dans la page via wordpress : expérience vidéo et expérience de playlist. Grâce à l'expérience vidéo, vous devez d'abord sélectionner l'expérience que vous souhaitez inclure. Dans la barre latérale, il y a une liste déroulante avec les expériences disponibles répertoriées.

Si vous cliquez sur « Insérer dans la publication » sans sélectionner de vidéos, vous insérez une expérience dans la page précédemment créée. Vous pouvez également sélectionner certaines vidéos, puis l'expérience en page que vous souhaitez inclure pour transformer ces vidéos en expérience. De même, grâce à l'expérience de playlist, sélectionnez une playlist et une expérience, puis insérez dans la publication. La liste de lecture sera convertie en une expérience.

Connecteur WordPress-Brightcove : Configuration

Cette rubrique traite de la configuration du connecteur Brightcove pour Wordpress.

Introduction

WordPress Brightcove Video Connector vous permet de gérer les vidéos et les lecteurs Brightcove Video Cloud dans WordPress, et d'intégrer facilement des vidéos dans les pages WordPress.

Connectez-vous au compte Brightcove

  1. Accédez aux paramètres Brightcove et cliquez sur Ajouter un compte Brightcove.
  2. Le nom source peut être n'importe quoi, mais utiliser le même nom que le nom du compte sur Brightcove facilitera l'identification.
  3. Pour trouver l'ID de compte, connectez-vous à Brightcove Studio et regardez dans le coin supérieur gauche de votre tableau de bord.
  4. Pour obtenir l'ID client et le secret client, vous devrez enregistrer une application. Accédez à l'authentification API et cliquez sur Ajouter une nouvelle application.
  5. Remplissez le nom et la brève description.
  6. Sélectionnez un compte. Sur la photo ci-dessous sont toutes les autorisations requises.
  7. Appuyez sur Enregistrer.
  8. L'ID client et le secret client s'affichent. Assurez-vous de copier et d'enregistrer Client Secret dans un emplacement sécurisé. Vous ne pourrez plus y accéder.
  9. Cliquez sur « Vérifier les informations d'identification.« Si vous réussissez, vos vidéos doivent être affichées.

Définir le profil d'ingest

Version 1.4.1 du connecteur et code dur précédent le profil d'ingest Video Cloud pour être « videocloud-default-v1".  Si le compte Brightcove est uniquement activé par Delivery dynamique ou CAE, ce profil d'ingeste n'existera pas et le téléchargement de la vidéo échouera.  Voici plusieurs options pour contourner ce comportement dans le connecteur.

 

  1. Créez un nouveau profil d'ingeste dans le compte Video Cloud avec le nom de profil par défaut utilisé dans le connecteur.  Accédez à Admin->Ingérer des profils et créez un nouveau profil ou dupliquez un profil existant et nommez-le « videocloud-default-v1 ».  

 

  1. Vous pouvez également modifier le nom du profil d'ingestion dans le code source du connecteur en déterminant d'abord quel profil d'ingeste existant sous Admin->Ingestion dans le compte Video Cloud que vous souhaitez utiliser. Cliquez sur Afficher le nom de l'API et notez le.
  1. Sur le serveur WordPress, modifiez wp-content/plugins/brightcove-video-connect/includes/api/class-bc-cms-api.php et modifiez le profil du nom de profil par défaut de « videocloud-default-v1 »' au nom de profil souhaité.  Par exemple, changez
    public function video_upload( $video_id, $video_url, $profile = 'videocloud-default-v1' ) {
    pour
    public function video_upload( $video_id, $video_url, $profile = 'multi-platform-standard-static' ) {
  1. Ou modifiez le connecteur pour utiliser le profil d'ingeste par défaut dans le compte Video Cloud. Pour ce faire, modifiez wp-content/plugins/brightcove-video-connect/includes/api/class-bc-cms-api.php et commentez la ligne :
    $data = array( 'profile' => sanitize_text_field( $profile ) );
    avec /* */.

Vous avez maintenant terminé la configuration du connecteur WordPress-BrightCove. Passez à Utilisation du connecteur.

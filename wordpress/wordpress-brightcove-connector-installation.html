Connecteur WordPress-Brightcove : Installation

Cette rubrique traite de l'installation du connecteur Brightcove pour Wordpress.

Introduction

WordPress Brightcove Video Connector vous permet de gérer les vidéos et les lecteurs Brightcove Video Cloud dans WordPress, et d'intégrer facilement des vidéos dans les pages WordPress.

Étapes d'installation

  1. Téléchargez le plug-in WordPress Brightcove Video Connect
  2. Décompressez le fichier et passez /brightcove-video-connect à /wordpress/wp-content/plugins
  3. Activez Brightcove Video Connect via le menu Plugins dans WordPress

Si le plugin Brightcove est visible dans la barre latérale, vous pouvez passer à Configuration.
